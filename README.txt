# 🐶 바둑이 주식 웹사이트 사용법

주인님! 웹사이트 코드를 모두 작성했어요. 
실시간으로 구글 시트 데이터를 가져오기 위해 딱 **한 가지 설정**이 필요해요.

## 1. 구글 시트 '웹에 게시' 설정하기 (필수!)

웹사이트가 주인님의 시트를 읽을 수 있도록 권한을 주는 과정입니다.

1. 주인님의 **구글 시트(주식종합_2026)**를 엽니다.
2. 상단 메뉴에서 **파일 (File)** > **공유 (Share)** > **웹에 게시 (Publish to web)**를 클릭합니다.
3. 팝업 창이 뜨면:
   - **[링크]** 탭에서
   - **전체 문서** 대신 **'Summary'** 시트를 선택합니다.
   - **웹 페이지** 대신 **'쉼표로 구분된 값(.csv)'**을 선택합니다.
   - **게시 (Publish)** 버튼을 누릅니다.
   - 생성된 **링크를 복사**해두세요! (이게 Summary 링크입니다)

4. 똑같은 방법으로 **'Holdings'** 시트도 링크를 만들어주세요.
   - 다시 **웹에 게시** 창에서
   - 이번엔 **'Holdings'** 시트를 선택하고, **'쉼표로 구분된 값(.csv)'**으로 설정합니다.
   - 이 링크도 복사하세요! (이게 Holdings 링크입니다)

## 2. 코드에 링크 넣기

1. 이 폴더에 있는 `script.js` 파일을 엽니다. (메모장이나 코드 에디터로)
2. 파일 맨 위에 있는 `CONFIG` 부분을 찾으세요.

```javascript
const CONFIG = {
    // ⚠️ 주인님, 이 주소를 실제 CSV 주소로 바꿔주세요!
    summaryURL: "여기에_Summary_링크_붙여넣기",
    holdingsURL: "여기에_Holdings_링크_붙여넣기" 
};
```

3. 복사해둔 링크를 각각 따옴표("") 안에 붙여넣고 저장해주세요.

## 3. 실행하기

- 이제 `index.html` 파일을 더블 클릭해서 열면 됩니다!
- 크롬, 사파리 등 웹 브라우저에서 바로 보실 수 있어요. 🐶

문제가 생기면 언제든 저를 불러주세요! 멍!
